# 프롬프트 변경 이력

> 챗봇/훈련 모드 프롬프트의 모든 변경을 기록.
> 정확도 하락 원인 추적 + 롤백 판단 근거.

---

## 작성 규칙

```
### vN | YYYY-MM-DD | 파일명 | 변경자
- **변경**: 무엇을 바꿨는지
- **이유**: 왜 바꿨는지
- **영향**: 어떤 동작이 달라지는지
- **커밋**: 해시
```
